How to install in Gmail

  
    
    
  1. Click Copy signature above.
    2. 
    
  2. In Gmail, click the ⚙ gear icon (top right) → See all settings.
    3. 
    
  3. Stay on the General tab → scroll to Signature.
    4. 
    
  4. Click + Create new (or select an existing signature to replace).
    5. 
    
  5. Click inside the signature editor box and paste (⌘V / Ctrl+V).
    6. 
    
  6. Under Signature defaults, set this signature for For new emails use and On reply/forward use.
    7. 
    
  7. Scroll to the bottom → Save Changes.
    8. 
  

  

    Mobile: Gmail's iOS/Android apps have their own signature field (Settings → your account → Signature settings). They don't sync with the web signature and only accept plain text — so use a short plain-text fallback there.


    Testing: send yourself a test from web Gmail to a different address before rolling out to the team. Check it lands correctly in (1) Gmail web inbox, (2) Gmail mobile preview.
